# 🧠 Python Desktop Mind Map Tool

A modern, interactive, and high-performance Desktop Mind Mapping application built using Python, **CustomTkinter**, and **Tkinter Canvas**. This tool allows users to visually organize thoughts, brainstorm ideas, and map out projects seamlessly with an infinite workspace feel.

---

## 🚀 Features

- **➕ Dynamic Node Creation:** Add new ideas (nodes) instantly with a single click.
- **✋ Infinite Canvas Panning:** Left-click and drag on any empty space to move the canvas in any direction.
- **🔍 Smart Zooming:** Smooth Zoom In (+) and Zoom Out (-) to manage small or large mind maps easily.
- **🖱️ Interactive Context Menu:** Right-click on any node to:
  - 📝 Edit node text dynamically.
  - 🎨 Change individual node colors via an advanced Color Picker.
  - 🔗 Link/Connect nodes together with beautifully colored lines.
  - 🗑️ Delete nodes and automatically clean up their connected lines.
- **💾 Project Save & Load:** Export your full mind map data into a lightweight `.json` file and reload it anytime to continue your work.
- **🔒 Security Lock:** Toggle the "Lock Map" switch to freeze the workspace into a read-only state, protecting your layout from accidental edits.
- **📸 High-Quality Export:** Save your final mind map as a **PNG Image** or a printable **PDF Document** with one click.
